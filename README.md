# Suivi du covoiturage à Angers

## Objectif

L'objectif de ce code est de visualiser les données de covoiturage recueillies à Angers.

Il sert de base pour le déploiement d'une app Streamlit : https://covoitangers.streamlit.app/

## Contexte

Le projet est porté par le Cerema.

Un capteur de covoiturage de la société Invision.ai a été installé le 18 février 2026 par la société Equans, sur la D523, dans le sens entrant vers Angers.
Voir le site sur Google Street View : https://maps.app.goo.gl/jFscxgeSwcsrVd9L9

Le Département du Maine et Loire est associé au déploiement du capteur pour une durée minimale d'un an.
